# Atalhos do Visual Studio Code





## 1. Atalhos Gerais

| Ação                     | Windows / Linux    | macOS              |
| ------------------------ | ------------------ | ------------------ |
| Abrir Paleta de Comandos | `Ctrl + Shift + P` | `Cmd + Shift + P`  |
| Abrir configurações      | `Ctrl + ,`         | `Cmd + ,`          |
| Novo arquivo             | `Ctrl + N`         | `Cmd + N`          |
| Abrir arquivo            | `Ctrl + O`         | `Cmd + O`          |
| Salvar arquivo           | `Ctrl + S`         | `Cmd + S`          |
| Salvar todos             | `Ctrl + K S`       | `Cmd + Option + S` |
| Fechar editor            | `Ctrl + W`         | `Cmd + W`          |
| Sair do VS Code          | `Ctrl + Q`         | `Cmd + Q`          |

------

## 2. Navegação entre Arquivos e Editores

| Ação                            | Windows / Linux    | macOS                  |
| ------------------------------- | ------------------ | ---------------------- |
| Alternar entre arquivos abertos | `Ctrl + Tab`       | `Cmd + Option + → / ←` |
| Ir para arquivo rapidamente     | `Ctrl + P`         | `Cmd + P`              |
| Ir para linha específica        | `Ctrl + G`         | `Cmd + G`              |
| Ir para símbolo no arquivo      | `Ctrl + Shift + O` | `Cmd + Shift + O`      |
| Ir para símbolo no workspace    | `Ctrl + T`         | `Cmd + T`              |

------

## 3. Edição de Código

| Ação                         | Windows / Linux       | macOS                    |
| ---------------------------- | --------------------- | ------------------------ |
| Comentar linha               | `Ctrl + /`            | `Cmd + /`                |
| Comentar bloco               | `Shift + Alt + A`     | `Shift + Option + A`     |
| Duplicar linha               | `Shift + Alt + ↓ / ↑` | `Shift + Option + ↓ / ↑` |
| Mover linha                  | `Alt + ↓ / ↑`         | `Option + ↓ / ↑`         |
| Apagar linha                 | `Ctrl + Shift + K`    | `Cmd + Shift + K`        |
| Formatar documento           | `Shift + Alt + F`     | `Shift + Option + F`     |
| Selecionar palavra           | `Ctrl + D`            | `Cmd + D`                |
| Selecionar todas ocorrências | `Ctrl + Shift + L`    | `Cmd + Shift + L`        |

## 4. Seleção e Cursores Múltiplos

| Ação                     | Windows / Linux      | macOS                  |
| ------------------------ | -------------------- | ---------------------- |

------

## Observações Importantes

- Todos os atalhos acima são **oficiais e configurados por padrão** no VS Code.
- Podem variar se o usuário:
  - Alterar o mapa de teclas
  - Usar extensões (ex: Vim, Emacs)
- É possível visualizar ou personalizar atalhos em:
  **File → Preferences → Keyboard Shortcuts**
